Description

Binary Classification of Images

Description

This project is comprised of an AI model for binary classification of images, the architecture applied was a Convolutional Neural Network and it was applied using TensorFlow and Keras. Techniques as early stopping and exponential decay for the learning rate were implemented to avoid overfitting and optimize the computational cost, respectively. An accuracy greater than 86% was reached.
